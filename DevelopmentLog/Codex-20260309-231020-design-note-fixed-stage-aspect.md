# 设计弹层首图定画布比例

时间：2026-03-09 23:10:20
执行者：Codex

## 本次改动

- 更新了 `src/components/DesignNoteModal.vue` 左侧图片展示逻辑。
- 现在进入弹层后，会先读取第一张图片的实际宽高比例。
- 左侧展示区会用第一张图的比例作为整组轮播的固定画布。
- 后续图片不再把容器撑来撑去，而是统一用 `contain` 方式适配进这个固定画布。
- 这样：
  - 第一张是横图时，后续竖图会在横向画布里居中显示
  - 第一张是竖图时，后续横图会在纵向画布里居中显示
- 同时增加了基于 `ResizeObserver` 的视口适配，保证弹层缩放时画布仍按固定比例居中收缩。
