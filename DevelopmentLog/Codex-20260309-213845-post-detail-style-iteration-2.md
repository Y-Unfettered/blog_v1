# 文章详情页样式二次调整

时间：2026-03-09 21:38:45
执行者：Codex

## 调整方向

- 放弃上一版偏“作品集展示”的重装饰风格，改成更克制的博客阅读页。
- 减少大面积发光、厚重卡片和强视觉特效，强调标题、摘要、正文和目录的层次。
- 把详情页重新收口到“编辑部 / 杂志式阅读”方向。

## 本次改动

- 重写 `src/views/PostDetailView.vue` 的结构和局部样式。
- 头部改成更简洁的标题区，只保留分类、置顶、摘要、时间信息和标签。
- 目录卡改成更轻的右侧粘性卡片，移动端改成胶囊式目录入口。
- 正文容器改成弱边框、弱背景的阅读区，减少视觉噪音。
- 正文内部的标题、段落、引用、代码块、表格和图片样式同步收敛，避免过度设计。
- 上下篇导航改成更平的阅读卡片，弱化夸张渐变和漂浮效果。

- `npm run build` 通过
