# 设计创作页沉浸式弹层交互

时间：2026-03-09 22:55:11
执行者：Codex

## 本次改动

- 为设计创作页新增了沉浸式弹层组件 `src/components/DesignNoteModal.vue`。
- 点击设计卡片后，不再直接进入普通文章详情页，而是在设计页内打开一个居中的浏览层。
- 弹层支持：
  - 左上角圆形关闭按钮
  - 点击遮罩空白区域关闭
  - `Esc` 关闭
  - 打开/关闭时的缩放过渡动画
  - 左侧图片区域点击切换轮播（如果后续数据里有多图会直接生效）
  - 右侧文字区域独立滚动
- 设计创作页 `src/views/DesignView.vue` 已改为使用这个本地弹层交互。
- `src/App.vue` 为设计页补传了 `tagName`，支持弹层里显示标签名。

## 当前数据说明

- 现有设计文章数据里只有 `cover`，没有额外多图。
- 所以现在轮播结构已经具备，但大多数文章当前只会显示单张封面。
- 后续如果你给设计文章补 `images` 数组，或者正文里插入多张 Markdown 图片，弹层左侧会自动按多图轮播工作。

## 验证结果
